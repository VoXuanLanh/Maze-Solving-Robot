# Maze-Solving-Robot
Robot sử dụng 3 cảm biến hồng ngoại. Nguyên tắc hoạt động là cảm biến sẽ phát tia hồng ngoại và nhận phản xạ từ vật thể phía trước. Các tín hiệu này được Arduino đọc liên tục, rồi từ đó quyết định hướng đi theo thuật toán Right-Hand Rule và có sự điều chỉnh mở rộng ( lùi ) trong thuật toán giải mê cung này.
